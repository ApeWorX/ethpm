class ethpm_types.contract_type.ContractType(*, contractName: str = None, sourceId: str = None, deploymentBytecode: ethpm_types.contract_type.Bytecode = None, runtimeBytecode: ethpm_types.contract_type.Bytecode = None, abi: List[Union[ethpm_types.abi.ConstructorABI, ethpm_types.abi.FallbackABI, ethpm_types.abi.ReceiveABI, ethpm_types.abi.MethodABI, ethpm_types.abi.EventABI, ethpm_types.abi.ErrorABI, ethpm_types.abi.StructABI, ethpm_types.abi.UnprocessedABI]] = [], sourcemap: ethpm_types.contract_type.SourceMap = None, userdoc: dict = None, devdoc: dict = None)

A serializable type representing the type of a contract. For example, if you define your contract as contract MyContract (in Solidity), then MyContract would be the type.

abi: List[Union[ethpm_types.abi.ConstructorABI, ethpm_types.abi.FallbackABI, ethpm_types.abi.ReceiveABI, ethpm_types.abi.MethodABI, ethpm_types.abi.EventABI, ethpm_types.abi.ErrorABI, ethpm_types.abi.StructABI, ethpm_types.abi.UnprocessedABI]]

The application binary interface to the contract.

property constructor: ethpm_types.abi.ConstructorABI

The constructor of the contract, if it has one. For example, your smart-contract (in Solidity) may define a constructor() public {}. This property contains information about the parameters needed to initialize a contract.

deployment_bytecode: Optional[ethpm_types.contract_type.Bytecode]

The bytecode for the ContractType.

property events: List[ethpm_types.abi.EventABI]

The events defined in a smart contract. :returns: List[ABI]

property fallback: ethpm_types.abi.FallbackABI

The fallback method of the contract, if it has one. A fallback method is external, has no name, arguments, or return value, and gets invoked when the user attempts to call a method that does not exist.

property mutable_methods: List[ethpm_types.abi.MethodABI]

The transaction-methods (stateful or payable methods) defined in a smart contract. :returns: List[ABI]

name: Optional[str]

The name of the contract type. The field is optional if ContractAlias is the same as ContractName.

runtime_bytecode: Optional[ethpm_types.contract_type.Bytecode]

The unlinked 0x-prefixed runtime portion of bytecode for this ContractType.

source_id: Optional[str]

The global source identifier for the source file from which this contract type was generated.

sourcemap: Optional[ethpm_types.contract_type.SourceMap]

The range of the source code that is represented by the respective node in the AST. NOTE: This is not part of the canonical EIP-2678 spec.

property view_methods: List[ethpm_types.abi.MethodABI]

The call-methods (read-only method, non-payable methods) defined in a smart contract. :returns: List[ABI]

class ethpm_types.contract_type.SourceMap(*, __root__: str)

As part of the AST output, the compiler provides the range of the source code that is represented by the respective node in the AST. This can be used for various purposes ranging from static analysis tools that report errors based on the AST and debugging tools that highlight local variables and their uses.

Solidity Doc.

parse() → Iterator[ethpm_types.contract_type.SourceMapItem]

Parses the source map string into a stream of SourceMapItem items. Useful for when parsing the map according to compiler-specific decompilation rules back to the source code language files.
