class ethpm_types.source.Checksum(*, algorithm: ethpm_types.utils.Algorithm, hash: ethpm_types.utils.Hex)

Checksum information about the contents of a source file.

algorithm: ethpm_types.utils.Algorithm

The algorithm used to generate the corresponding hash. Possible algorithms include, but are not limited to sha3, sha256, md5, keccak256.

hash: ethpm_types.utils.Hex

The hash of a source files contents generated with the corresponding algorithm.

calculate_checksum(algorithm: ethpm_types.utils.Algorithm = Algorithm.MD5) → ethpm_types.source.Checksum

Compute the checksum of the Source object. Will short-circuit to content identifier if using content-addressed file references Fails if content isn’t available locally or by fetching.

Parameters

algorithm (Optional[Algorithm]) – The algorithm to use to compute the checksum with. Defaults to MD5.

Returns

Checksum

checksum: Optional[ethpm_types.source.Checksum]

Hash of the source file. Per EIP-2678, this field is only necessary if source must be fetched.

content: Optional[str]

Inlined contract source.

content_is_valid() → bool

Return if content is corrupted. Will never be corrupted if content is locally available. If content is referenced by content addressed identifier, will not be corrupted either. If referenced from a server URL, then checksum must be present and will be validated against.

Returns

bool

fetch_content() → str

Fetch the content for the given Source object. Loads resource at urls into content if not available.

str
